Principales commandes en SQL, principes de base : (format glossaire)

SQL Select	Exo1	Insert the missing statement to get all the columns from the Customers table. 								SELECT * FROM Customers; 			rem: où '*' signifie tout.

		Exo2	Write a statement that will select the City column from the Customers table.								SELECT City FROM Customers;			rem: sélectionne colonne 																											'City' de la table 																											'Customers'.

		Exo3	Select all the different values from the Country column in the Customers table.								SELECT DISTINCT Country FROM Customers;		rem: sélectionne 																												différentes valeurs 																											colonne 'Country' de 																											la table 																												'Customers'.

SQL Where	Exo1	Select all records where the City column has the value "Berlin".									SELECT * FROM Customers WHERE City = 'Berlin';	rem:	-

		Exo2	Select all records where the CustomerID column has the value 32.									SELECT * FROM Customers WHERE CustomerID = 32;	rem:	-

SQL Order By	Exo1	Select all records from the Customers table, sort the result alphabetically by the column City.						SELECT * FROM Customers ORDER BY City;		rem: trie éléments colonne 																											'City' 																													alphabétiquement.

		Exo2	Select all records from the Customers table, sort the result reversed alphabetically by the column City.				SELECT * FROM Customers ORDER BY City DESC;	rem: DESC = descending

		Exo3	Select all records from the Customers table, sort the result alphabetically, column Country, column City.				SELECT * FROM Customers
																				ORDER BY Country, City				rem: tri 2 clés
